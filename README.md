<h1 align="center">Mini Vue</h1>

Mini Vue 是一个基于 @vue/reactivity 的小程序框架，它简单，强大，高性能。 你可以使用 Vue 3 的响应式数据 + 组合式 API 开发小程序，从此开发小程序也将是一种享受。
